# 🚀 Инструкция по запуску Red Pulse Mini App

## Быстрый старт (5 минут)

### Шаг 1: Установка зависимостей

Откройте терминал в папке проекта и выполните:

```bash
pip install -r requirements.txt
```

### Шаг 2: Настройка .env

Файл `.env` уже настроен для локальной разработки:

```env
BOT_TOKEN=8472410996:AAHOFRseu_tOHRatvvemo6by5ihaHBJOF6o
DATABASE_URL=sqlite+aiosqlite:///./redpulse.db
WEBAPP_URL=http://127.0.0.1:8000/webapp
ADMIN_USERNAME=admin
ADMIN_PASSWORD=redpulse2026
```

### Шаг 3: Запуск сервера (Mini App + Админка)

Откройте первый терминал:

```bash
python run_server.py
```

Вы увидите:
```
🚀 Запуск сервера Red Pulse Mini App...
📡 Админ-панель: http://127.0.0.1:8000
🎮 Mini App: http://127.0.0.1:8000/webapp
```

### Шаг 4: Настройка ngrok для HTTPS

Telegram требует HTTPS для Mini App. Используем ngrok:

1. **Скачайте ngrok**: https://ngrok.com/download
2. **Установите и зарегистрируйтесь** (бесплатно)
3. **Запустите во втором терминале**:

```bash
ngrok http 8000
```

4. **Скопируйте HTTPS URL** (например: `https://abc123.ngrok-free.app`)

### Шаг 5: Настройка в @BotFather

1. Откройте Telegram → @BotFather
2. Отправьте `/mybots`
3. Выберите вашего бота
4. **Bot Settings** → **Menu Button** → **Configure Menu Button**
5. Отправьте URL: `https://abc123.ngrok-free.app/webapp` (ваш URL из ngrok + /webapp)
6. Введите название: `🎮 Открыть игру`

### Шаг 6: Запуск бота

Откройте третий терминал:

```bash
python run_bot.py
```

## 🔧 Решение проблем

### Mini App не открывается
1. Проверьте, что сервер запущен (`python run_server.py`)
2. Проверьте, что ngrok активен
3. Убедитесь, что URL в BotFather содержит `/webapp` на конце

### Ошибка "Network Error"
- Обновите страницу Mini App
- Проверьте подключение к интернету
- Перезапустите ngrok

### Бот не отвечает
1. Проверьте, что `python run_bot.py` запущен
2. Проверьте BOT_TOKEN в `.env`

### Данные не сохраняются
- Убедитесь, что файл `redpulse.db` существует
- Проверьте права доступа к папке

## 🌐 Production (размещение в интернете)

### Вариант 1: Vercel (бесплатно)

1. Создайте аккаунт на https://vercel.com
2. Установите Vercel CLI: `npm i -g vercel`
3. Разместите проект: `vercel --prod`

### Вариант 2: Railway (бесплатно)

1. https://railway.app
2. Подключите GitHub репозиторий
3. Добавьте переменные окружения

### Вариант 3: Свой сервер (VPS)

1. Купите домен и SSL сертификат
2. Установите Docker или Python
3. Разместите проект
4. Настройте Nginx как reverse proxy

## 💡 Советы

1. **Не закрывайте терминалы** — сервер и бот должны работать постоянно
2. **ngrok бесплатен 2 часа** — для долгой работы купите тариф или используйте свой домен
3. **Сохраняйте URL ngrok** — при каждом запуске новый URL (на бесплатном тарифе)
4. **Тестируйте на себе** — сначала проверьте всё сами, потом приглашайте пользователей
