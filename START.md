# 🚀 ЗАПУСК RED PULSE MINI APP (ЛОКАЛЬНО)

## ⚡ Самый быстрый способ

Просто запустите файл **`start_local.bat`** (двойной клик).

Откроются два окна:
- 🟢 **Сервер** - Mini App + Админка + API
- 🔵 **Бот** - Telegram бот

## 📡 Адреса для тестирования

| Что открыть | URL |
|-------------|-----|
| **Mini App (тест)** | http://127.0.0.1:8000/webapp/test |
| Mini App | http://127.0.0.1:8000/webapp |
| Админка | http://127.0.0.1:8000 |

## 🎮 Как тестировать

### 1. В браузере (рекомендуется)

1. Откройте **http://127.0.0.1:8000/webapp/test**
2. Тестовый пользователь уже настроен (ID: 123456789)
3. Кликайте, покупайте, играйте!

### 2. В Telegram

1. Откройте вашего бота
2. Нажмите `/game`
3. Telegram покажет превью Mini App

## 🔧 Внесение изменений

### Правка Mini App

```
webapp/index.html → Сохранить → Обновить страницу (F5)
```

### Правка API

```
webapp_routes.py → Сохранить → Сервер перезагрузится сам
```

### Правка бота

```
bot/handlers/*.py → Сохранить → Перезапустить бота
```

## 🛠️ Структура проекта

```
redpulse-bot/
├── start_local.bat      ← Запуск одним кликом
├── run_server.py        ← Сервер (FastAPI)
├── run_bot.py           ← Бот (aiogram)
├── dev.py               ← Dev режим (оба в одном)
│
├── webapp/
│   └── index.html       ← Mini App (frontend)
│
├── bot/
│   ├── handlers/
│   │   ├── start.py     ← /start, /game
│   │   ├── profile.py   ← /profile
│   │   ├── tasks.py     ← /tasks
│   │   ├── support.py   ← /support
│   │   └── announcements.py ← /announcements
│   └── keyboards.py     ← Кнопки
│
├── webapp_routes.py     ← API для Mini App
├── admin.py             ← Админка + сервер
├── main.py              ← Бот (main)
│
└── .env                 ← Настройки
```

## 🎯 Что работает в Mini App

- ✅ **Кликер** - заработок монет
- ✅ **Бусты** - энергия, сила, автокликер
- ✅ **Казино** - кости, слоты, блэкджек
- ✅ **Магазин** - кейсы, скины, аватары
- ✅ **Задания** - выполнение и награда
- ✅ **Профиль** - статистика игрока

## 🤖 Что работает в боте

- ✅ `/start` - приветствие
- ✅ `/profile` - краткий профиль
- ✅ `/game` - открыть Mini App
- ✅ `/tasks` - задания
- ✅ `/support` - поддержка
- ✅ `/announcements` - анонсы

## 💡 Советы для разработки

1. **Используйте `webapp/test`** - не нужен Telegram
2. **Откройте DevTools (F12)** - видны ошибки
3. **Сервер перезагружается сам** - reload=True
4. **Бот перезапускайте вручную** - Ctrl+C → запустить

## 🐛 Проблемы

| Проблема | Решение |
|----------|---------|
| Порт 8000 занят | Закройте другое приложение или измените порт |
| Бот не запускается | Проверьте `.env` и BOT_TOKEN |
| Mini App не грузится | Проверьте, что сервер запущен |
| Ошибка БД | Удалите `redpulse.db` для сброса |
