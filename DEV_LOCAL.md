# 🔧 Локальная разработка Red Pulse Mini App

## Быстрый старт (Windows)

### Вариант 1: Одним кликом

Просто запустите файл:

```
start_local.bat
```

Откроются два окна:
- **Сервер** (FastAPI + Mini App + Админка)
- **Бот** (Telegram бот)

### Вариант 2: Вручную

**Терминал 1 - Сервер:**
```bash
python run_server.py
```

**Терминал 2 - Бот:**
```bash
python run_bot.py
```

## 📡 Адреса для тестирования

| Компонент | URL |
|-----------|-----|
| Mini App | http://127.0.0.1:8000/webapp |
| Тест Mini App | http://127.0.0.1:8000/webapp/test |
| Админ-панель | http://127.0.0.1:8000 |
| API (user) | http://127.0.0.1:8000/api/user/123456789 |

## 🧪 Тестирование Mini App

### В браузере (без Telegram)

1. Откройте: **http://127.0.0.1:8000/webapp/test**
2. Используется тестовый пользователь (ID: 123456789)
3. Все функции работают как в Telegram

### В Telegram (локально)

1. Откройте бота в Telegram
2. Отправьте `/game`
3. Telegram откроет превью Mini App

⚠️ **Важно:** Telegram может не открыть http:// ссылку. Для полноценной работы нужен HTTPS (ngrok).

## 🎮 Тестовый пользователь

Для локальной разработки используется тестовый пользователь:

- **ID:** 123456789
- **Имя:** Тестовый пользователь
- **Username:** test_user

Все данные сохраняются в БД и будут видны после перезапуска.

## 🔧 Внесение изменений

### Изменения в Mini App

1. Правьте `webapp/index.html`
2. Обновите страницу в браузере (F5)
3. Сервер автоматически перезагружается (reload=True)

### Изменения в API

1. Правьте `webapp_routes.py`
2. Сервер автоматически перезагружается
3. Обновите страницу Mini App

### Изменения в боте

1. Правьте файлы в `bot/handlers/`
2. Перезапустите бота (Ctrl+C → `python run_bot.py`)

## 🚀 Переход на HTTPS (ngrok)

Когда готовы к тестированию в Telegram:

```bash
# Терминал 3
ngrok http 8000
```

Скопируйте HTTPS URL и обновите в @BotFather.

## 💡 Советы

1. **Используйте браузер** для быстрой отладки Mini App
2. **DevTools (F12)** для просмотра ошибок
3. **Тестовый пользователь** не требует Telegram
4. **Автосохранение** каждые 30 секунд
5. **Логи в консоль** для отладки

## 📞 Проблемы

| Проблема | Решение |
|----------|---------|
| Порт 8000 занят | Измените порт в `run_server.py` |
| Бот не запускается | Проверьте BOT_TOKEN в `.env` |
| Mini App пустой | Откройте консоль браузера (F12) |
| Ошибка БД | Удалите `redpulse.db` и перезапустите |
